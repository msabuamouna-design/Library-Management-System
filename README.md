# Library-Management-System 
A professional Python application designed to manage library books efficiently using an **SQLite** database.

##  Features
* **Secure Login:** Admin authentication system.
* **Book Management:** Add, and store book details (Title, Author, ISBN, Price).
* **Database Integration:** Automatic data persistence using SQLite3.
* **User-friendly CLI:** Clean terminal interface for data entry.

##  Technologies Used
* **Language:** Python 3.x
* **Database:** SQLite3
* **Tools:** VS Code, GitHub
